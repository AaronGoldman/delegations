How to authorize an agent

  1. Have the agent send a request to this endpoint (without cookies).
  2. The agent receives a 401 response containing a delegation_url.
  3. Open the delegation_url in a browser and approve the grant.
  4. Tell the agent to retry — it will now be authorized.
